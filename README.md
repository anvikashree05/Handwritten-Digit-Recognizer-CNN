# Handwritten Digit Recognizer using CNN

## Project Overview

This project uses a Convolutional Neural Network (CNN) built with TensorFlow and Keras to recognize handwritten digits from images.

The model is trained on the MNIST dataset containing 70,000 handwritten digit images and achieves approximately 99% validation accuracy.

## Features

* Handwritten digit recognition (0–9)
* CNN-based image classification
* Custom image prediction support
* Model saving and loading
* High accuracy (~99%)

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Pillow

## Dataset

MNIST Dataset

* Training Images: 60,000
* Testing Images: 10,000

## Results

* Training Accuracy: 99.38%
* Validation Accuracy: 99.13%
